# Docker-xcbuild
Using Facebook xcbuild inside docker

## Download
`docker pull mattpaletta/xcbuild:latest`

## Usage
`docker run -it mattpaletta/xcbuild bash`
From there, use `./xcbuild` to build your iOS project.
